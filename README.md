**The More You Look, The Less You See**

 
An interactive virtual reality installation about attention, agency, and the cost of giving our focus away. 
Built in Unity for  Meta Quest 3, it places a single user in a dark void where abstract glowing stimuli emerge outside their periphery and call for their attention. The more the user engages by looking at the stimuli, the more stimuli distractions they generate. The experience is designed to become increasingly overstimulating, directly proportional to the user's ability to reject the temptation of new stimuli. 


Spoken instructions explaining how to escape the experience restart every time the user's gaze wanders, in order to complete the experience, they must make it to the end of the instructions without falling into the feedback loop. 

The piece reframes attention as a coveted resource and proposes that deliberate restraint, not looking, can be an avenue of freedom.

Input: head-position tracking as a proxy for attention 
Stimuli: glowing peripheral forms with unique colour, movement, sound, and arrival cues; looking at one spawns more

Narrative: a spoken script of instructions that restarts on every glance, so distraction obstructs the answer

Subtitles: fixed-in-view text that resets alongside the voice

Overload: a soft reset with a warning if the user becomes too distracted

Resolution: earned only by hearing the narrative through uninterrupted and staying still, making restraint the win condition

Attention map: a live, fluid gaze trail with dendritic fixation nodes, output to a second screen for an audience

Engine: Unity 6 (URP) version 6000.3, built for Meta Quest 3 via OpenXR


repository holds the Unity scripts, with process documented in the accompanying blog, report, and video
